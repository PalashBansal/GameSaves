Steps to use this game profile for Bridge Constructor Portal(BCP)

Steps(This is for Windows OS, Paths will differ per OS):
1. Rename existing folder %USERPROFILE%\AppData\LocalLow\ClockStone Software GmbH\Bridge Constructor Portal\
2. Place downloaded "Bridge Constructor Portal" folder in %USERPROFILE%\AppData\LocalLow\ClockStone Software GmbH\

Game profiles are-
"%USERPROFILE%\AppData\LocalLow\ClockStone Software GmbH\Bridge Constructor Portal\ga.sqlite3"
"%USERPROFILE%\AppData\LocalLow\ClockStone Software GmbH\Bridge Constructor Portal\Profile.dat"
